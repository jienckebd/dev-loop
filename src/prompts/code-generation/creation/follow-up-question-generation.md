# Follow-Up Question Generation Prompt

Generate follow-up questions based on previous answers to dive deeper into unclear areas.

## Context

You are generating follow-up questions based on previous Q&A to clarify remaining gaps.

## Instructions

1. Analyze previous questions and answers to identify gaps
2. Generate questions that dive deeper into unclear areas
3. Ask about specific implementation details, dependencies, edge cases
4. Generate 2-5 follow-up questions per iteration
5. Avoid repeating already answered questions

## Output Format

Return questions as JSON array (same structure as initial questions).
