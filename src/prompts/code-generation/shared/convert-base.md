# Convert Mode Base Prompt

Base prompt for converting planning documents to PRD sets.

## Mode: Convert

Convert a planning document (Claude Code planning doc format) into a fully executable dev-loop PRD set.

## Key Requirements

1. Parse planning document structure accurately
2. Generate comprehensive PRD set with all phases and tasks
3. Include schema definitions based on codebase patterns
4. Generate test specifications for all requirements
5. Add framework-specific configuration
6. Ensure 100% executability

## Process

1. Analyze planning document structure
2. Extract phases, tasks, and requirements
3. Analyze codebase for patterns and conventions
4. Generate enhancements (schemas, tests, config)
5. Validate executability
6. Generate PRD set files
