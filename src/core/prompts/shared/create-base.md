# Create Mode Base Prompt

Base prompt for creating PRD sets from user prompts through iterative questions.

## Mode: Create

Create a new PRD set from a simple user prompt through iterative clarification questions.

## Key Requirements

1. Generate clarifying questions based on user prompt
2. Process answers iteratively
3. Generate comprehensive PRD draft from collected answers
4. Apply refinement iterations
5. Ensure 100% executability

## Process

1. Analyze user prompt and codebase
2. Generate initial clarifying questions (3-10 questions)
3. Ask questions and collect answers iteratively
4. Generate follow-up questions based on answers
5. Generate initial PRD draft from all answers
6. Apply refinement iterations (schemas, tests, config)
7. Generate PRD set files
